Prediksi Usia Teripang (Class: Holothuroidea)

Aplikasi untuk memprediksi usia Teripang berdasarkan panjang tubuh menggunakan model pertumbuhan Von Bertalanffy

Dibuat oleh: Andy Rasyadi, S.Pi., M.Si. Dosen Manajemen Sumberdaya Perairan, Fakultas Kelautan dan Perikanan, Universitas Udayana

Parameter Model

Database ilmiah akan dicocokkan dengan nama spesies yang diinput
Rumus Von Bertalanffy: Lt = L∞ × (1 - e-K(t - t₀))
Parameter L∞ dan K didapatkan dari penelitian sebelumnya. Nilai default yang ditampilkan adalah contoh nilai yang umum digunakan untuk Teripang yang anda pilih.
Belum ada file dipilih
File CSV harus berisi satu kolom panjang Teripang dalam satuan cm. Contoh format: 12.5, 15.2, 18.7, ...
